Apa yang Akan Dibahas?

Dirancang untuk pengembang web dan edukator yang ingin membangun platform pembelajaran digital dengan standar industri.

Sesi 01

Konsep Headless CMS

Memahami perbedaan CMS tradisional vs arsitektur Headless — dan mengapa pemisahan backend konten dari frontend UI sangat penting untuk performa dan skalabilitas.

Sesi 02

Setup Sanity Studio

Langkah demi langkah mengonfigurasi schema data di Sanity Studio untuk menyusun struktur modul online, bab, hingga rich text content.

Sesi 03

Integrasi Frontend

Menarik data menggunakan bahasa query GROQ dan menampilkannya di frontend modern — live demo dengan Tailwind CSS dan vanilla JS.

Sesi 04

Deployment

Strategi deployment terpisah untuk frontend dan Sanity Studio menggunakan platform cloud modern — gratis hingga skala enterprise.
