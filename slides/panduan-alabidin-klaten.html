CMS (Content Management System)

Wordpress digunakan sebagai cms untuk membangun web ini. Harapannya familiaritasnya bisa memudahkan ketika membuat post dan page.

Dibantu plugin Elementor untuk membangun sebuah halaman kustom diluar bawaan tema.
